技术领先点 1：生成列 + 自动索引

这是云器 JSON 能力的核心技术优势，不是"功能够用"。

-- 传统 JSON 查询：全表扫描，慢 SELECT * FROM products WHERE attributes['ram_gb'] > 8; -- 扫描 500TB -- 云器方案：创建生成列，自动索引 ALTER TABLE products ADD COLUMN ram_gb INT GENERATED ALWAYS AS ( CAST(attributes['ram_gb'] AS INT) ); CREATE INDEX idx_ram ON products(ram_gb); -- 自动索引 -- 查询：走索引，秒级 SELECT * FROM products WHERE ram_gb > 8; -- 数据裁剪，只读相关块

技术原理

  • 生成列：从 JSON 中提取字段，虚拟列（不占存储）
  • 自动索引：云器自动为生成列创建索引，无需手动维护
  • 数据裁剪：查询时自动使用索引，跳过无关数据块
  • 透明优化：业务 SQL 不变，性能提升 10-100 倍
数美科技验证：

500TB JSON 日志，传统 Spark 查询需要数十分钟。
云器生成列 + 自动索引后，同样查询秒级响应。

这不是"能用"，是技术代差。
