根据你的需求，以下是创建物化视图 dws.mv_daily_revenue 的完整 SQL，每小时自动刷新，统计每天每个 region 的 revenue 总和，数据来自 dwd.orders_clean。

## 创建物化视图

```sql
CREATE OR REPLACE MATERIALIZED VIEW dws.mv_daily_revenue
  COMMENT '每日各 region 收入汇总'
  REFRESH AUTO EVERY '1 hours'
  VCLUSTER = default_ap
AS
SELECT
  DATE(created_at) AS day,
  region,
  SUM(amount) AS revenue
FROM dwd.orders_clean
GROUP BY 1, 2;
```

## 验证

```sql
-- 确认物化视图已创建
SHOW TABLES WHERE is_materialized_view = true;

-- 查看结构
DESC MATERIALIZED VIEW dws.mv_daily_revenue;
```

## 手动触发刷新（可选）

```sql
REFRESH MATERIALIZED VIEW dws.mv_daily_revenue;
```

## 运维操作（可选）

```sql
-- 暂停自动刷新
ALTER MATERIALIZED VIEW dws.mv_daily_revenue SUSPEND;

-- 恢复自动刷新
ALTER MATERIALIZED VIEW dws.mv_daily_revenue RESUME;
```

## 说明

- REFRESH AUTO EVERY '1 hours'：每小时自动全量刷新（ClickZetta 特有语法，必须带引号和 AUTO 关键字）
- VCLUSTER = default_ap：使用默认计算集群执行刷新
- 聚合逻辑：按 DATE(created_at) 和 region 分组，对 amount 求和得到每天每个 region 的 revenue 总和
- 物化视图为全量刷新，不支持增量；如需秒/分钟级增量刷新，应改用 Dynamic Table
