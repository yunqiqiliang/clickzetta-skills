## 创建动态表 dwd.orders_clean

根据你的需求，以下是创建动态表的完整 SQL：

```sql
CREATE OR REPLACE DYNAMIC TABLE dwd.orders_clean
  TARGET_LAG = '1 minutes'
  VCLUSTER = default_ap
AS
SELECT *
FROM ods.orders
WHERE amount > 0;
```

**参数说明：**
- `TARGET_LAG = '1 minutes'`：每分钟自动增量刷新一次
- `VCLUSTER = default_ap`：使用 default_ap 计算集群
- `WHERE amount > 0`：过滤掉 amount 为 0 或负数的数据

**执行后验证：**

```sql
-- 确认动态表已创建
SHOW TABLES WHERE is_dynamic = true;

-- 查看刷新历史，确认刷新正常运行
SHOW DYNAMIC TABLE REFRESH HISTORY dwd.orders_clean LIMIT 5;
```

**运维操作（按需使用）：**

```sql
-- 暂停动态表（如集群维护时）
ALTER DYNAMIC TABLE dwd.orders_clean SUSPEND;

-- 恢复动态表
ALTER DYNAMIC TABLE dwd.orders_clean RESUME;
```

> 注意：ClickZetta 使用 `VCLUSTER` 而非 `WAREHOUSE` 指定计算集群，这是与 Snowflake 的关键语法差异。
