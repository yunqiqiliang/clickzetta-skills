以下是创建动态表 dwd.orders_clean 的 SQL 语句：

```sql
CREATE OR REPLACE DYNAMIC TABLE dwd.orders_clean
  TARGET_LAG = '1 minutes'
  VCLUSTER = default_ap
AS
SELECT *
FROM ods.orders
WHERE amount > 0;
```

执行后可通过以下 SQL 验证：

```sql
-- 确认动态表已创建
SHOW TABLES IN dwd WHERE is_dynamic = true;

-- 查看刷新历史
SHOW DYNAMIC TABLE REFRESH HISTORY dwd.orders_clean LIMIT 5;
```

说明：
- TARGET_LAG = '1 minutes'：每分钟自动增量刷新一次
- VCLUSTER = default_ap：使用 default_ap 计算集群执行刷新任务
- WHERE amount > 0：过滤掉 amount 为 0 或负数的数据
